## 簡介

Playwright 支援多種語言，這些語言共享相同的底層實作。所有自動化瀏覽器的核心功能在所有語言中都受到支援，而測試生態系統的整合則有所不同。根據您的經驗、對其測試生態系統的熟悉程度以及您的專案限制來選擇語言。為了獲得最佳體驗，請選擇我們為每種語言推薦的測試執行器。

## JavaScript 和 TypeScript

Playwright for Node.js 附帶自己的[測試執行器](https://playwright.tw/docs/running-tests)，提供出色的平行化機制、螢幕截圖斷言、HTML 報告器、自動追蹤等功能。
* [文件](https://playwright.tw/docs/intro)
* [GitHub repo](https://github.com/microsoft/playwright)

## Python

Playwright [Pytest 外掛](https://playwright.tw/python/docs/test-runners)是執行端到端測試的推薦方式。它提供情境隔離、在多種瀏覽器組態設定上執行以及更多現成功能。
* [文件](https://playwright.tw/python/docs/intro)
* [GitHub repo](https://github.com/microsoft/playwright-python)

## Java

您可以根據您的專案需求選擇任何測試框架，例如 JUnit 或 TestNG。
* [文件](https://playwright.tw/java/docs/intro)
* [GitHub repo](https://github.com/microsoft/playwright-java)

## .NET

Playwright for .NET 附帶 MSTest、NUnit、xUnit 和 xUnit v3 [基底類別](https://playwright.tw/dotnet/docs/test-runners)用於撰寫端到端測試。
* [文件](https://playwright.tw/dotnet/docs/intro)
* [GitHub repo](https://github.com/microsoft/playwright-dotnet)
