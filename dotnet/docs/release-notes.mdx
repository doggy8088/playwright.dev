#### ğŸ“ æ–°çš„ [**Locators API**](./api/class-locator)

The difference between the [Locator](./api/class-locator) å’Œ [ElementHandle](./api/class-elementhandle) is that the latter points to a particular element, while [Locator](./api/class-locator) captures the logic of how to retrieve that element.

