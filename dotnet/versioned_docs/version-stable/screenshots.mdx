## 簡介

這裡有一個快速捕捉螢幕截圖並將其保存到文件中的方法:

```csharp
await Page.ScreenshotAsync(new()
{
    Path = "screenshot.png",
});
```

[Screenshots API](./api/class-page#page-screenshot) 接受許多參數來設定影像格式、裁剪區域、品質等。請務必查看。

## 全頁截圖

整頁截圖是一個完整可滾動頁面的截圖，就像你有一個非常高的螢幕，頁面可以完全適應它。

```csharp
await Page.ScreenshotAsync(new()
{
    Path = "screenshot.png",
    FullPage = true,
});
```

## 截圖並儲存至 Buffer

與其寫入檔案，你可以獲取截圖的 Buffer 物件並進行後處理或將其傳遞給其他工具比較像素級別的差異。

```csharp
var bytes = await page.ScreenshotAsync();
Console.WriteLine(Convert.ToBase64String(bytes));
```

## 元素截圖

有時候將單一元素截圖是有用的。

```csharp
await page.Locator(".header").ScreenshotAsync(new() { Path = "screenshot.png" });
```
