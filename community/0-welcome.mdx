# 歡迎

歡迎來到 Playwright 社群。我們很高興您加入我們。在社群區段中，請查看我們的影片區塊，您可以看到[會議演講](./conference-videos)、[直播](./live-streams)、[功能介紹影片](./feature-videos)和[發佈影片](./release-videos)。

## 大使

查看我們的[大使頁面](./ambassadors)，了解這些傑出的人們如何創建 Playwright 內容並與社群分享。

## GitHub

我們喜歡星星，所以請務必[在 GitHub 上給我們加星](https://github.com/microsoft/playwright)。

請針對以下情況建立議題:

- [錯誤回報](https://github.com/microsoft/playwright/issues/new?assignees=&labels=&template=bug.md&title=%5BBUG%5D)
- [功能請求](https://github.com/microsoft/playwright/issues/new?assignees=&labels=&template=feature_request.md&title=%5BFeature%5D)
- [回報回歸](https://github.com/microsoft/playwright/issues/new?assignees=&labels=&template=regression.md&title=%5BREGRESSION%5D%3A+)
- [回報安全漏洞](https://github.com/microsoft/playwright/security/policy)

## 貢獻

如果您想要為 Playwright 做出貢獻，請查看我們的[貢獻指南](https://github.com/microsoft/playwright/blob/main/CONTRIBUTING.md)。

## 社群 Discord

加入我們的社群 [Discord 伺服器](https://aka.ms/playwright/discord)，與其他使用 Playwright 的開發者交流，在我們的 ['help-playwright'](https://discord.com/channels/807756831384403968/1054804523652231198) 論壇中提問，分享您的文章和影片，並參加 Playwright 舞台上的直播活動。

## 社群 LinkedIn

在 [LinkedIn](https://www.linkedin.com/company/playwrightweb/) 上加入我們的社群，與其他開發者聯繫並聽取 Playwright 的最新消息。

## Stack Overflow {#stack-overflow}

## YouTube
