

# 歡迎

歡迎來到 Playwright 社群。我們很高興你在這裡。在我們的社群部分，看看我們的影片部分，觀看有關[會議演講](./conference-videos)、[直播](./live-streams)、[精選影片](./feature-videos)和[發行影片](./release-videos)的影片。

## 技術傳教士

查看我們的 [技術傳教士頁面](./ambassadors) 來看看那些建立 Playwright 內容並與社群分享的了不起的人們。

## GitHub

我們喜歡星星，所以請確保你[在 GitHub 上給我們加星](https://github.com/microsoft/playwright)。

請建立一個問題如下:

- [錯誤回報](https://github.com/microsoft/playwright/issues/new?assignees=&labels=&template=bug.md&title=%5BBUG%5D)
- [功能請求](https://github.com/microsoft/playwright/issues/new?assignees=&labels=&template=feature_request.md&title=%5BFeature%5D)
- [回報回歸](https://github.com/microsoft/playwright/issues/new?assignees=&labels=&template=regression.md&title=%5BREGRESSION%5D%3A+)
- [回報安全漏洞](https://github.com/microsoft/playwright/security/policy)

## 貢獻

請查看我們的[貢獻指南](https://github.com/microsoft/playwright/blob/main/CONTRIBUTING.md) 如果你想為 Playwright 做出貢獻。

## 社群 Discord

加入我們的社群 [Discord Server](https://aka.ms/playwright/discord) 來與其他使用 Playwright 的開發者聯繫，在我們的['help-playwright'](https://discord.com/channels/807756831384403968/1054804523652231198)論壇中提問，分享你的文章和影片，並參加 Playwright 舞台上的現場活動。

## Community LinkedIn

加入我們的 [LinkedIn](https://www.linkedin.com/company/playwrightweb/) 社群，與其他開發者聯繫並聽取有關 Playwright 的最新消息。

## 堆疊 Overflow {#stack-overflow}

閱讀這些標記為 **playwright** 的[現有問題](https://stackoverflow.com/tags/playwright)或[提出你自己的問題](https://stackoverflow.com/questions/ask?tags=playwright)！

## YouTube

請查看我們的 [YouTube channel](https://www.youtube.com/channel/UC46Zj8pDH5tDosqm1gd7WTg) 以獲取入門系列、精選影片和新版本發布。

## 部落格

請關注我們在 dev.to 上的 [Blog](https://dev.to/playwright)，以獲取 Playwright 的官方文章。

## 新聞 {#news}

如需有關 Playwright 的最新消息，請[關注 Twitter 上的 **@playwrightweb**](https://twitter.com/playwrightweb)。
