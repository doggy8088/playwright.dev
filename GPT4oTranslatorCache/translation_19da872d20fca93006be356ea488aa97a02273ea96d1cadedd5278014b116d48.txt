- `args` [List]\[[str]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-args"/><a href="#browser-type-launch-option-args" class="list-anchor">#</a>

  :::warning
  使用自訂瀏覽器 args 需自行承擔風險，因為其中一些可能會破壞 Playwright 功能。
  :::

  傳遞給瀏覽器實例的額外參數。可以在[這裡](https://peter.sh/experiments/chromium-command-line-switches/)找到 Chromium 標誌的列表。
- `channel` [str] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-channel"/><a href="#browser-type-launch-option-channel" class="list-anchor">#</a>

  瀏覽器分發頻道。支持的值有 "chrome", "chrome-beta", "chrome-dev", "chrome-canary", "msedge", "msedge-beta", "msedge-dev", "msedge-canary"。閱讀更多關於使用[Google Chrome 和 Microsoft Edge](../browsers.mdx#google-chrome--microsoft-edge)的資訊。
- `chromium_sandbox` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-chromium-sandbox"/><a href="#browser-type-launch-option-chromium-sandbox" class="list-anchor">#</a>

  啟用 Chromium 沙盒。預設為 `false`。
- `devtools` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-devtools"/><a href="#browser-type-launch-option-devtools" class="list-anchor">#</a>

  :::warning[Deprecated]
  使用[偵錯工具](../debug.mdx)代替。
  :::


  **僅限 Chromium** 是否為每個標籤頁自動打開開發者工具面板。如果此選項為 `true`，則 `headless` 選項將設置為 `false`。
- `downloads_path` [Union]\[[str], [pathlib.Path]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-downloads-path"/><a href="#browser-type-launch-option-downloads-path" class="list-anchor">#</a>

  如果指定，接受的下載將下載到此目錄。否則，將創建臨時目錄並在瀏覽器關閉時刪除。在任何情況下，當創建它們的瀏覽器上下文關閉時，下載將被刪除。
- `env` [Dict]\[[str], [str] | [float] | [bool]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-env"/><a href="#browser-type-launch-option-env" class="list-anchor">#</a>

  指定對瀏覽器可見的環境變量。預設為 `process.env`。
- `executable_path` [Union]\[[str], [pathlib.Path]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-executable-path"/><a href="#browser-type-launch-option-executable-path" class="list-anchor">#</a>

  要運行的瀏覽器可執行文件的路徑，而不是捆綁的那個。如果 `executable_path` 是相對路徑，則相對於當前工作目錄解析。請注意，Playwright 只適用於捆綁的 Chromium, Firefox 或 WebKit，使用需自行承擔風險。
- `firefox_user_prefs` [Dict]\[[str], [str] | [float] | [bool]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-firefox-user-prefs"/><a href="#browser-type-launch-option-firefox-user-prefs" class="list-anchor">#</a>

  Firefox 用戶偏好設置。了解更多關於 Firefox 用戶偏好設置的資訊在[`about:config`](https://support.mozilla.org/en-US/kb/about-config-editor-firefox)。
- `handle_sighup` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-handle-sighup"/><a href="#browser-type-launch-option-handle-sighup" class="list-anchor">#</a>

  在 SIGHUP 上關閉瀏覽器進程。預設為 `true`。
- `handle_sigint` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-handle-sigint"/><a href="#browser-type-launch-option-handle-sigint" class="list-anchor">#</a>

  在 Ctrl-C 上關閉瀏覽器進程。預設為 `true`。
- `handle_sigterm` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-handle-sigterm"/><a href="#browser-type-launch-option-handle-sigterm" class="list-anchor">#</a>

  在 SIGTERM 上關閉瀏覽器進程。預設為 `true`。
- `headless` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-headless"/><a href="#browser-type-launch-option-headless" class="list-anchor">#</a>

  是否以 headless 模式運行瀏覽器。更多細節請參考[Chromium](https://developers.google.com/web/updates/2017/04/headless-chrome)和[Firefox](https://developer.mozilla.org/en-US/docs/Mozilla/Firefox/Headless_mode)。預設為 `true`，除非 `devtools` 選項為 `true`。
- `ignore_default_args` [bool] | [List]\[[str]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-ignore-default-args"/><a href="#browser-type-launch-option-ignore-default-args" class="list-anchor">#</a>

  如果 `true`，Playwright 不會傳遞其自身的配置 args，只使用 `args` 中的那些。如果給定一個數組，則過濾掉給定的預設參數。危險選項；請謹慎使用。預設為 `false`。
- `proxy` [Dict] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-proxy"/><a href="#browser-type-launch-option-proxy" class="list-anchor">#</a>
  - `server` [str]

    用於所有請求的代理。支持 HTTP 和 SOCKS 代理，例如 `http://myproxy.com:3128` 或 `socks5://myproxy.com:3128`。簡寫 `myproxy.com:3128` 被視為 HTTP 代理。
  - `bypass` [str] *(optional)*

    可選的逗號分隔域名以繞過代理，例如 `".com, chromium.org, .domain.com"`。
  - `username` [str] *(optional)*

    如果 HTTP 代理需要身份驗證，則使用的可選用戶名。
  - `password` [str] *(optional)*

    如果 HTTP 代理需要身份驗證，則使用的可選密碼。

  網絡代理設置。
- `slow_mo` [float] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-slow-mo"/><a href="#browser-type-launch-option-slow-mo" class="list-anchor">#</a>

  以指定的毫秒數減慢 Playwright 操作。這樣可以看到發生了什麼。
- `timeout` [float] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-timeout"/><a href="#browser-type-launch-option-timeout" class="list-anchor">#</a>

  等待瀏覽器實例啟動的最長時間（毫秒）。預設為 `30000`（30 秒）。傳遞 `0` 以禁用超時。
- `traces_dir` [Union]\[[str], [pathlib.Path]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-traces-dir"/><a href="#browser-type-launch-option-traces-dir" class="list-anchor">#</a>

  如果指定，則將痕跡保存到此目錄。
