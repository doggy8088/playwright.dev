  - `Env` [IDictionary]?&lt;[string], [string]&gt; *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-env"/><a href="#browser-type-launch-option-env" class="list-anchor">#</a>

    要運行的瀏覽器可執行文件的路徑，而不是捆綁的那個。如果 `executablePath` 是相對路徑，則相對於當前工作目錄解析。請注意，Playwright 僅適用於捆綁的 Chromium、Firefox 或 WebKit，使用需自行承擔風險。
  - `FirefoxUserPrefs` [IDictionary]?&lt;[string], [object]&gt; *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-firefox-user-prefs"/><a href="#browser-type-launch-option-firefox-user-prefs" class="list-anchor">#</a>

  - `HandleSIGHUP` [bool]? *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-handle-sighup"/><a href="#browser-type-launch-option-handle-sighup" class="list-anchor">#</a>

  - `IgnoreAllDefaultArgs` [bool]? *(optional)* <font size="2">新增於: v1.9</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-ignore-all-default-args"/><a href="#browser-type-launch-option-ignore-all-default-args" class="list-anchor">#</a>

    如果 `true`，Playwright 不會傳遞其自己的配置參數，只使用 `args` 中的參數。危險選項；請謹慎使用。預設為 `false`。
  - `IgnoreDefaultArgs` [IEnumerable]?&lt;[string]&gt; *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-ignore-default-args"/><a href="#browser-type-launch-option-ignore-default-args" class="list-anchor">#</a>

  - `Proxy` Proxy? *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-launch-option-proxy"/><a href="#browser-type-launch-option-proxy" class="list-anchor">#</a>
    - `Server` [string]

      用於所有請求的代理。支持 HTTP 和 SOCKS 代理，例如 `http://myproxy.com:3128` 或 `socks5://myproxy.com:3128`。簡短形式 `myproxy.com:3128` 被視為 HTTP 代理。
    - `Bypass` [string]? *(optional)*

      可選的逗號分隔域名以繞過代理，例如 `".com, chromium.org, .domain.com"`。
    - `Username` [string]? *(optional)*

      如果 HTTP 代理需要身份驗證，則使用的可選用戶名。
    - `Password` [string]? *(optional)*

      如果 HTTP 代理需要身份驗證，則使用的可選密碼。
