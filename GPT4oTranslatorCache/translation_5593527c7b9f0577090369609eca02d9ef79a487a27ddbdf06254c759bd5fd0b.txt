- `options` `Route.FetchOptions` *(optional)*
  - `setHeaders` [Map]<[String], [String]> *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="route-fetch-option-headers"/><a href="#route-fetch-option-headers" class="list-anchor">#</a>
    
    如果設定，則更改請求 HTTP headers。Header 值將被轉換為字串。
  - `setMaxRedirects` [int] *(optional)* <font size="2">Added in: v1.31</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="route-fetch-option-max-redirects"/><a href="#route-fetch-option-max-redirects" class="list-anchor">#</a>
    
    將自動跟隨的請求重定向的最大數量。如果超過此數量，將拋出錯誤。預設為 `20`。傳遞 `0` 以不跟隨重定向。
  - `setMaxRetries` [int] *(optional)* <font size="2">Added in: v1.46</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="route-fetch-option-max-retries"/><a href="#route-fetch-option-max-retries" class="list-anchor">#</a>
    
    應重試網路錯誤的最大次數。目前僅重試 `ECONNRESET` 錯誤。不會根據 HTTP 響應代碼重試。如果超過限制，將拋出錯誤。預設為 `0` - 不重試。
  - `setMethod` [String] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="route-fetch-option-method"/><a href="#route-fetch-option-method" class="list-anchor">#</a>
    
    如果設定，則更改請求方法（例如 GET 或 POST）。
  - `setPostData` [String] | [byte&#91;&#93;] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="route-fetch-option-post-data"/><a href="#route-fetch-option-post-data" class="list-anchor">#</a>
    
    如果設定，則更改請求的 post 資料。
  - `setTimeout` [double] *(optional)* <font size="2">Added in: v1.33</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="route-fetch-option-timeout"/><a href="#route-fetch-option-timeout" class="list-anchor">#</a>
    
    請求超時時間（毫秒）。預設為 `30000`（30 秒）。傳遞 `0` 以禁用超時。
  - `setUrl` [String] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="route-fetch-option-url"/><a href="#route-fetch-option-url" class="list-anchor">#</a>
    
    如果設定，則更改請求 URL。新 URL 必須與原始 URL 擁有相同的協議。
