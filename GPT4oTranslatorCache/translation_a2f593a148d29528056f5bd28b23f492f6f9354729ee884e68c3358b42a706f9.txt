以下將說明如何將 Playwright 與 [Microsoft Edge WebView2](https://docs.microsoft.com/en-us/microsoft-edge/webview2/) 搭配使用。WebView2 是一個 WinForms 控制項，將使用 Microsoft Edge 來渲染網頁內容。它是 Microsoft Edge 瀏覽器的一部分，適用於 Windows 10 和 Windows 11。Playwright 可用於自動化 WebView2 應用程式，並可用於測試 WebView2 中的網頁內容。為了連接到 WebView2，Playwright 使用 [BrowserType.ConnectOverCDPAsync()](https://api/class-browsertype.mdx#browser-type-connect-over-cdp)，它通過 Chrome DevTools Protocol (CDP) 進行連接。
