  - `Animations` `enum ScreenshotAnimations { Disabled, Allow }?` *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="element-handle-screenshot-option-animations"/><a href="#element-handle-screenshot-option-animations" class="list-anchor">#</a>

    當設置為 `"disabled"` 時，停止 CSS 動畫、CSS 過渡和 Web 動畫。動畫根據其持續時間得到不同的處理：
    * 有限動畫將快速轉到完成，因此它們會觸發 `transitionend` 事件。
    * 無限動畫將取消到初始狀態，然後在截圖後重新播放。

    預設為 `"allow"`，保持動畫不變。
  - `Caret` `enum ScreenshotCaret { Hide, Initial }?` *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="element-handle-screenshot-option-caret"/><a href="#element-handle-screenshot-option-caret" class="list-anchor">#</a>

    當設置為 `"hide"` 時，截圖將隱藏文本插入符號。當設置為 `"initial"` 時，文本插入符號行為不會改變。預設為 `"hide"`。

    指定遮罩元素的覆蓋框的顏色，使用 [CSS 顏色格式](https://developer.mozilla.org/en-US/docs/Web/CSS/color_value)。預設顏色為粉紅色 `#FF00FF`。

  - `Quality` [int]? *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="element-handle-screenshot-option-quality"/><a href="#element-handle-screenshot-option-quality" class="list-anchor">#</a>

    圖像的品質，介於 0-100 之間。不適用於 `png` 圖像。
  - `Scale` `enum ScreenshotScale { Css, Device }?` *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="element-handle-screenshot-option-scale"/><a href="#element-handle-screenshot-option-scale" class="list-anchor">#</a>

    當設置為 `"css"` 時，截圖將每個 css 像素對應一個像素。對於高分辨率設備，這將保持截圖較小。使用 `"device"` 選項將產生每個設備像素對應一個像素，因此高分辨率設備的截圖將是兩倍或更大。

    預設為 `"device"`。
