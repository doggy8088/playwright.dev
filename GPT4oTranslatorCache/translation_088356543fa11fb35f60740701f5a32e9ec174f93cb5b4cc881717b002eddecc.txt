* 如果有服務工作者阻礙了你的操作，你現在可以使用新的上下文選項 `serviceWorkers` 輕鬆禁用它:

  ```ts title="playwright.config.ts"
  export default {
    use: {
      serviceWorkers: 'block',
    }
  };
  ```

* 使用 `.zip` 路徑作為 `recordHar` 上下文選項會自動壓縮生成的 HAR:

  ```ts
  const context = await browser.newContext({
    recordHar: {
      path: 'github.har.zip',
    }
  });
  ```

* 如果你打算手動編輯 HAR，考慮使用 `"minimal"` HAR 錄製模式，該模式僅記錄重播所需的必要資訊:

  ```ts
  const context = await browser.newContext({
    recordHar: {
      path: 'github.har',
      mode: 'minimal',
    }
  });
  ```

* Playwright 現在可以在 Ubuntu 22 amd64 和 Ubuntu 22 arm64 上執行。我們還發布了新的 docker 映像 `mcr.microsoft.com/playwright:v1.34.0-jammy`。
