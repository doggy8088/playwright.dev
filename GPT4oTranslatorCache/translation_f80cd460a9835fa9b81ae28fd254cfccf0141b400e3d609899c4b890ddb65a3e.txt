以下將解釋如何使用 Playwright 搭配 [Microsoft Edge WebView2](https://docs.microsoft.com/en-us/microsoft-edge/webview2/)。WebView2 是一個 WinForms 控制項，會使用 Microsoft Edge 來呈現網頁內容。它是 Microsoft Edge 瀏覽器的一部分，適用於 Windows 10 和 Windows 11。Playwright 可以用來自動化 WebView2 應用程式，並用來測試 WebView2 中的網頁內容。為了連接到 WebView2，Playwright 使用 [browserType.connectOverCDP()](https://api/class-browsertype.mdx#browser-type-connect-over-cdp)，該方法通過 Chrome DevTools Protocol (CDP) 進行連接。
