## 方法

### connect {#browser-type-connect}

<font size="2" style={{position: "relative", top: "-20px"}}>Added before v1.9</font><x-search>browserType.connect</x-search>

此方法將 Playwright 連接到透過 Node.js 中的 `BrowserType.launchServer` 建立的現有瀏覽器實例。

:::note

連接的 Playwright 實例的主要和次要版本需要與啟動瀏覽器的 Playwright 版本相符（1.2.3 → 與 1.2.x 相容）。
:::

**使用方式**

```python
browser_type.連接(ws_endpoint)
browser_type.連接(ws_endpoint, **kwargs)
```

**參數**
- `ws_endpoint` [str] <font size="2">Added in: v1.10</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-選項-ws-endpoint"/><a href="#browser-type-connect-option-ws-endpoint" class="list-anchor">#</a>
  
  要連接的 Playwright 瀏覽器 WebSocket 端點。您可以透過 `BrowserServer.wsEndpoint` 取得此端點。
- `expose_network` [str] *(optional)* <font size="2">Added in: v1.37</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-選項-expose-網路"/><a href="#browser-type-connect-option-expose-網路" class="list-anchor">#</a>
  
  此選項將連接用戶端可用的網路公開給要連接的瀏覽器。由以逗號分隔的規則清單組成。
  
  可用規則：
  1. 主機名稱樣式，例如：`example.com`、`*.org:99`、`x.*.y.com`、`*foo.org`。
  1. IP 位址字面值，例如：`127.0.0.1`、`0.0.0.0:99`、`[::1]`、`[0:0::1]:99`。
  1. `<loopback>` 符合本機迴環介面：`localhost`、`*.localhost`、`127.0.0.1`、`[::1]`。
  
  一些常見範例：
  1. `"*"` 公開所有網路。
  1. `"<loopback>"` 公開 localhost 網路。
  1. `"*.test.internal-domain,*.staging.internal-domain,<loopback>"` 公開測試/測試環境部署和 localhost。
- `headers` [Dict]\[[str], [str]\] *(optional)* <font size="2">Added in: v1.11</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-選項-headers"/><a href="#browser-type-connect-option-headers" class="list-anchor">#</a>
  
  與 web socket 連接請求一起傳送的額外 HTTP 標頭。選擇性。
- `slow_mo` [float] *(optional)* <font size="2">Added in: v1.10</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-選項-slow-mo"/><a href="#browser-type-connect-option-slow-mo" class="list-anchor">#</a>
  
  將 Playwright 操作減慢指定的毫秒數。這樣您可以看到正在發生的事情，很有用。預設值為 0。
- `timeout` [float] *(optional)* <font size="2">Added in: v1.10</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-選項-timeout"/><a href="#browser-type-connect-option-timeout" class="list-anchor">#</a>
  
  等待建立連接的最大時間（毫秒）。預設值為 `0`（無逾時）。

**回傳值**
- [Browser]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-return"/><a href="#瀏覽器-type-連接-return" class="list-anchor">#</a>

### connect_over_cdp {#瀏覽器-type-連接-over-cdp}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>browserType.connect_over_cdp</x-search>

此方法使用 Chrome DevTools Protocol 將 Playwright 連接到現有的瀏覽器實例。

預設瀏覽器情境可透過 [browser.contexts](/api/class-browser.mdx#browser-contexts) 存取。

:::note

透過 Chrome DevTools Protocol 連接僅支援基於 Chromium 的瀏覽器。
:::

:::note
此連接的保真度遠低於透過 [browser_type.connect()](/api/class-browsertype.mdx#browser-type-connect) 的 Playwright 協定連接。如果您遇到問題或嘗試使用進階功能，您可能會想要使用 [browser_type.connect()](/api/class-browsertype.mdx#browser-type-connect)。
:::

**使用方式**

<Tabs
  groupId="python-flavor"
  defaultValue="sync"
  values={[
    {label: 'Sync', value: 'sync'},
    {label: 'Async', value: 'async'}
  ]
}>
<TabItem value="sync">

```py
瀏覽器 = playwright.chromium.connect_over_cdp("http://localhost:9222")
default_context = 瀏覽器.contexts[0]
page = default_context.pages[0]
```

</TabItem>
<TabItem value="async">

```py
瀏覽器 = await playwright.chromium.connect_over_cdp("http://localhost:9222")
default_context = 瀏覽器.contexts[0]
page = default_context.pages[0]
```

</TabItem>
</Tabs>

**參數**
- `endpoint_url` [str] <font size="2">Added in: v1.11</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-over-cdp-選項-endpoint-url"/><a href="#browser-type-connect-over-cdp-option-endpoint-url" class="list-anchor">#</a>
  
  要連接的 CDP WebSocket 端點或 http url。例如 `http://localhost:9222/` 或 `ws://127.0.0.1:9222/devtools/browser/387adf4c-243f-4051-a181-46798f4a46f4`。
- `headers` [Dict]\[[str], [str]\] *(optional)* <font size="2">Added in: v1.11</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-over-cdp-選項-headers"/><a href="#browser-type-connect-over-cdp-option-headers" class="list-anchor">#</a>
  
  與連接請求一起傳送的額外 HTTP 標頭。選擇性。
- `slow_mo` [float] *(optional)* <font size="2">Added in: v1.11</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-over-cdp-選項-slow-mo"/><a href="#browser-type-connect-over-cdp-option-slow-mo" class="list-anchor">#</a>
  
  將 Playwright 操作減慢指定的毫秒數。這樣您可以看到正在發生的事情，很有用。預設值為 0。
- `timeout` [float] *(optional)* <font size="2">Added in: v1.11</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-over-cdp-選項-timeout"/><a href="#browser-type-connect-over-cdp-option-timeout" class="list-anchor">#</a>
  
  等待建立連接的最大時間（毫秒）。預設值為 `30000`（30 秒）。傳遞 `0` 可停用逾時。

**回傳值**
- [Browser]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-連接-over-cdp-return"/><a href="#瀏覽器-type-連接-over-cdp-return" class="list-anchor">#</a>

### launch {#browser-type-launch}

<font size="2" style={{position: "relative", top: "-20px"}}>Added before v1.9</font><x-search>browserType.啟動</x-search>

回傳瀏覽器實例。

**使用方式**

您可以使用 [ignore_default_args](/api/class-browsertype.mdx#browser-type-launch-option-ignore-default-args) 從預設參數中過濾掉 `--mute-audio`：

<Tabs
  groupId="python-flavor"
  defaultValue="sync"
  values={[
    {label: 'Sync', value: 'sync'},
    {label: 'Async', value: 'async'}
  ]
}>
<TabItem value="sync">

```py
瀏覽器 = playwright.chromium.啟動( # or "firefox" or "webkit".
    ignore_default_args=["--mute-audio"]
)
```

</TabItem>
<TabItem value="async">

```py
瀏覽器 = await playwright.chromium.啟動( # or "firefox" or "webkit".
    ignore_default_args=["--mute-audio"]
)
```

</TabItem>
</Tabs>

> **僅限 Chromium** Playwright 也可以用來控制 Google Chrome 或 Microsoft Edge 瀏覽器，但它與所綁定的 Chromium 版本配合得最好。不保證它會與任何其他版本配合。使用 [executable_path](/api/class-browsertype.mdx#browser-type-launch-option-executable-path) 選項時要極其謹慎。
>
> 如果偏好 Google Chrome（而非 Chromium），建議使用 [Chrome Canary](https://www.google.com/chrome/browser/canary.html) 或 [Dev Channel](https://www.chromium.org/getting-involved/dev-channel) 建置版本。
>
> 像 Google Chrome 和 Microsoft Edge 這樣的原廠瀏覽器適合需要專有媒體編解碼器進行視訊播放的測試。請參閱[這篇文章](https://www.howtogeek.com/202825/what%E2%80%99s-the-difference-between-chromium-and-chrome/)了解 Chromium 和 Chrome 之間的其他差異。[這篇文章](https://chromium.googlesource.com/chromium/src/+/lkgr/docs/chromium_browser_vs_google_chrome.md)描述了 Linux 使用者的一些差異。

**參數**
- `args` [List]\[[str]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-args"/><a href="#browser-type-launch-option-args" class="list-anchor">#</a>
  
  :::warning
  
  自行承擔使用自訂瀏覽器參數的風險，因為其中一些可能會破壞 Playwright 功能。
  :::
  
  傳遞給瀏覽器實例的額外參數。Chromium 旗標清單可在[這裡](https://peter.sh/experiments/chromium-command-line-switches/)找到。
- `channel` [str] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-通道"/><a href="#browser-type-launch-option-通道" class="list-anchor">#</a>
  
  瀏覽器分發通道。
  
  使用 "chromium" 來[選擇使用新的無頭模式](../browsers.mdx#chromium-new-headless-mode)。
  
  使用 "chrome"、"chrome-beta"、"chrome-dev"、"chrome-canary"、"msedge"、"msedge-beta"、"msedge-dev" 或 "msedge-canary" 來使用品牌[Google Chrome 和 Microsoft Edge](../browsers.mdx#google-chrome--microsoft-edge)。
- `chromium_sandbox` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-chromium-sandbox"/><a href="#browser-type-launch-option-chromium-sandbox" class="list-anchor">#</a>
  
  啟用 Chromium 沙盒。預設值為 `false`。
- `devtools` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-devtools"/><a href="#browser-type-launch-option-devtools" class="list-anchor">#</a>
  
  :::warning[已棄用]
  請改用[偵錯工具](../debug.mdx)。
  :::
  
  
  **僅限 Chromium** 是否為每個分頁自動開啟開發者工具面板。如果此選項為 `true`，[headless](/api/class-browsertype.mdx#browser-type-launch-option-headless) 選項將設定為 `false`。
- `downloads_path` [Union]\[[str], [pathlib.路徑]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-downloads-路徑"/><a href="#browser-type-launch-option-downloads-路徑" class="list-anchor">#</a>
  
  如果指定，已接受的下載會下載到此目錄。否則，會建立暫存目錄，並在瀏覽器關閉時刪除。無論哪種情況，下載都會在建立它們的瀏覽器情境關閉時被刪除。
- `env` [Dict]\[[str], [str] | [float] | [bool]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-env"/><a href="#browser-type-launch-option-env" class="list-anchor">#</a>
  
  指定瀏覽器可見的環境變數。預設值為 `process.env`。
- `executable_path` [Union]\[[str], [pathlib.路徑]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-executable-path"/><a href="#browser-type-launch-option-executable-path" class="list-anchor">#</a>
  
  要執行的瀏覽器可執行檔路徑，用來替代綁定的瀏覽器。如果 [executable_path](/api/class-browsertype.mdx#browser-type-launch-option-executable-path) 是相對路徑，則會相對於目前工作目錄進行解析。請注意，Playwright 僅與綁定的 Chromium、Firefox 或 WebKit 配合運作，請自行承擔使用風險。
- `firefox_user_prefs` [Dict]\[[str], [str] | [float] | [bool]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-firefox-user-prefs"/><a href="#browser-type-launch-option-firefox-user-prefs" class="list-anchor">#</a>
  
  Firefox 使用者偏好設定。深入了解 Firefox 使用者偏好設定，請參考 [`about:config`](https://support.mozilla.org/en-US/kb/about-config-editor-firefox)。
  
  您也可以透過 `PLAYWRIGHT_FIREFOX_POLICIES_JSON` 環境變數提供自訂 [`policies.json` 檔案](https://mozilla.github.io/policy-templates/)的路徑。
- `handle_sighup` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-handle-sighup"/><a href="#browser-type-launch-option-handle-sighup" class="list-anchor">#</a>

- `handle_sigint` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-handle-sigint"/><a href="#browser-type-launch-option-handle-sigint" class="list-anchor">#</a>

- `handle_sigterm` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-handle-sigterm"/><a href="#browser-type-launch-option-handle-sigterm" class="list-anchor">#</a>
  
  Close 瀏覽器process on SIGTERM. 預設為`true`.
- `headless` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-headless"/><a href="#browser-type-launch-option-headless" class="list-anchor">#</a>
  
  是否run 瀏覽器 in 無頭 mode. 更多詳細資訊for [Chromium](https://developers.google.com/web/updates/2017/04/headless-chrome) and [Firefox](https://hacks.mozilla.org/2017/12/using-headless-mode-in-firefox/). 預設為`true` unless the [devtools](/api/class-browsertype.mdx#browser-type-launch-option-devtools) 選項 is `true`.
- `ignore_default_args` [bool] | [List]\[[str]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-ignore-預設-args"/><a href="#browser-type-launch-option-ignore-預設-args" class="list-anchor">#</a>

- `slow_mo` [float] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-slow-mo"/><a href="#browser-type-launch-option-slow-mo" class="list-anchor">#</a>

- `timeout` [float] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-timeout"/><a href="#browser-type-launch-option-timeout" class="list-anchor">#</a>

- `traces_dir` [Union]\[[str], [pathlib.路徑]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-選項-traces-dir"/><a href="#browser-type-launch-option-traces-dir" class="list-anchor">#</a>
  
  若指定，追蹤檔案將儲存到此目錄.

### launch_persistent_context {#瀏覽器-type-啟動-持久-情境}

<font size="2" style={{position: "relative", top: "-20px"}}>Added before v1.9</font><x-search>browserType.launch_persistent_context</x-search>

回傳持久瀏覽器情境實例。

啟動使用位於 [user_data_dir](/api/class-browsertype.mdx#browser-type-launch-persistent-context-option-user-data-dir) 的持久儲存的瀏覽器，並回傳唯一的情境。關閉此情境會自動關閉瀏覽器。

**使用方式**

```python
browser_type.launch_persistent_context(user_data_dir)
browser_type.launch_persistent_context(user_data_dir, **kwargs)
```

**參數**
- `user_data_dir` [Union]\[[str], [pathlib.路徑]\]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-user-data-dir"/><a href="#browser-type-launch-persistent-context-option-user-data-dir" class="list-anchor">#</a>
  
  到...的路徑a User Data 目錄, which stores 瀏覽器 工作階段 data like cookies and local 儲存. Pass an empty string to create a temporary 目錄.
  
  更多詳細資訊for [Chromium](https://chromium.googlesource.com/chromium/src/+/master/docs/user_data_dir.md#introduction) and [Firefox](https://wiki.mozilla.org/Firefox/CommandLineOptions#User_profile). Chromium's user data 目錄 is the **parent** 目錄 of the "設定檔 路徑" seen at `chrome://version`.
  
  請注意browsers do not allow launching multiple instances with the same User Data 目錄.
  
  :::warning
  
  Chromium/Chrome: Due to recent Chrome policy changes, automating the 預設 Chrome user 設定檔 is not supported. Pointing `userDataDir` to Chrome's main "User Data" 目錄 (the 設定檔 used for your regular browsing) may result in pages not loading or 瀏覽器exiting. Create and use a separate 目錄 (例如, an empty 資料夾) as your automation 設定檔 instead. See https://developer.chrome.com/blog/remote-debugging-port for details.
  :::
  
- `accept_downloads` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-accept-downloads"/><a href="#browser-type-launch-persistent-context-option-accept-downloads" class="list-anchor">#</a>
  
  是否automatically 下載 all the attachments. 預設為`true` where all the downloads are accepted.
- `args` [List]\[[str]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-args"/><a href="#browser-type-launch-persistent-context-option-args" class="list-anchor">#</a>
  
  :::warning
  
  Use custom 瀏覽器 args at your own risk, as some of them may break Playwright functionality.

  - `height` [int]
    
    影片畫面高度。
  
  錄製影片的尺寸。如果未指定，大小將等於縮放後適合 800x800 的 `viewport`。如果未明確配置 `viewport`，影片大小預設為 800x450。如有需要，每個頁面的實際畫面會縮放以適合指定大小。
- `reduced_motion` "reduce" | "no-preference" | "null" *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-reduced-motion"/><a href="#browser-type-launch-persistent-context-option-reduced-motion" class="list-anchor">#</a>
  
  模擬 `'prefers-reduced-motion'` 媒體功能，支援的值為 `'reduce'`、`'no-preference'`。更多詳細資訊請參考 [page.emulate_media()](/api/class-page.mdx#page-emulate-media)。傳遞 `'null'` 會重設模擬為系統預設值。預設值為 `'no-preference'`。
- `screen` [Dict] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-螢幕"/><a href="#browser-type-launch-persistent-context-option-螢幕" class="list-anchor">#</a>
  - `width` [int]
    
    頁面寬度（像素）。
  - `height` [int]
    
    page height in pixels.
  
  模擬網頁內可用的一致視窗螢幕大小 via `window.screen`. Is only used when the [viewport](/api/class-browsertype.mdx#browser-type-launch-persistent-context-option-viewport) is set.
- `service_workers` "allow" | "block" *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-service-workers"/><a href="#browser-type-launch-persistent-context-option-service-workers" class="list-anchor">#</a>
  
  是否允許網站註冊 Service Worker. 預設為`'allow'`.
  * `'allow'`: [Service Workers](https://developer.mozilla.org/en-US/docs/Web/API/Service_Worker_API) can be registered.
  * `'block'`: Playwright will block all registration of Service Workers.
- `slow_mo` [float] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-slow-mo"/><a href="#browser-type-launch-persistent-context-option-slow-mo" class="list-anchor">#</a>
  
  Slows down Playwright operations by the specified amount of milliseconds. Useful so that you can see what is going on.
- `strict_selectors` [bool] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-strict-selectors"/><a href="#browser-type-launch-persistent-context-option-strict-selectors" class="list-anchor">#</a>
  
  若設定為 `true`，會為此情境啟用嚴格選擇器模式. In the strict selectors mode all operations on selectors that imply single target DOM element will throw when more than one element matches the selector. 此選項does not affect any Locator APIs (Locators are always strict). 預設為`false`. See [Locator] to 深入了解about the strict mode.
- `timeout` [float] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-timeout"/><a href="#browser-type-launch-persistent-context-option-timeout" class="list-anchor">#</a>
  
  等待瀏覽器實例啟動的最長時間（以毫秒為單位）. 預設為`30000` (30 seconds). Pass `0` to disable 逾時.
- `timezone_id` [str] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-timezone-id"/><a href="#browser-type-launch-persistent-context-option-timezone-id" class="list-anchor">#</a>
  
  變更情境的時區. See [ICU's metaZones.txt](https://cs.chromium.org/chromium/src/third_party/icu/source/data/misc/metaZones.txt?rcl=faee8bc70570192d82d2978a71e2a615788597d1) for a list of supported 時區 IDs. 預設為the system 時區.
- `traces_dir` [Union]\[[str], [pathlib.路徑]\] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-traces-dir"/><a href="#browser-type-launch-persistent-context-option-traces-dir" class="list-anchor">#</a>
  
  若指定，追蹤檔案將儲存到此目錄.
- `user_agent` [str] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-user-agent"/><a href="#browser-type-launch-persistent-context-option-user-agent" class="list-anchor">#</a>
  
  此情境要使用的特定 user agent.
- `viewport` [NoneType] | [Dict] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="browser-type-啟動-持久-情境-選項-檢視區"/><a href="#browser-type-launch-persistent-context-option-檢視區" class="list-anchor">#</a>
  - `width` [int]
    
    page width in pixels.
  - `height` [int]
    
    page height in pixels.
  
  Sets a consistent 檢視區 for each page. 預設為 1280x720 檢視區. `no_viewport` disables the fixed 檢視區. 深入了解about [viewport emulation](../emulation.mdx#viewport).
