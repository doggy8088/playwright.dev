`Accessibility` 類別提供檢查 Chromium 無障礙樹的方法。無障礙樹被輔助技術使用，例如[螢幕閱讀器](https://en.wikipedia.org/wiki/Screen_reader)或[開關存取](https://en.wikipedia.org/wiki/Switch_access)。

無障礙是一個非常特定於平台的功能。在不同的平台上，有不同的螢幕閱讀器，它們可能有截然不同的輸出。

Chromium、Firefox 和 WebKit 的渲染引擎都有「無障礙樹」的概念，然後將其轉換為不同的特定於平台的 API。`Accessibility` 命名空間提供對此無障礙樹的存取。

當從內部瀏覽器 AX 樹轉換為特定於平台的 AX 樹或由輔助技術本身進行轉換時，大部分無障礙樹會被過濾掉。預設情況下，Playwright 會嘗試近似此過濾，只公開樹中「有趣」的節點。

## 已棄用

### snapshot {#accessibility-snapshot}

<font size="2" style={{position: "relative", top: "-20px"}}>Added before v1.9</font><x-search>accessibility.snapshot</x-search>

:::warning 已棄用

此方法已棄用。如果您需要測試頁面無障礙性，請使用其他函式庫，例如 [Axe](https://www.deque.com/axe/)。請參閱我們的 Node.js [指南](https://playwright.dev/docs/accessibility-testing)以了解如何與 Axe 整合。

:::


擷取無障礙樹的目前狀態。回傳的物件代表頁面的根無障礙節點。

:::note

Chromium 無障礙樹包含在大多數平台和大多數螢幕閱讀器上未使用的節點。Playwright 也會捨棄它們以獲得更易於處理的樹，除非將 [interesting_only](/api/class-accessibility.mdx#accessibility-snapshot-option-interesting-only) 設定為 `false`。
:::

        if found_node:
            return found_node
    return None

**參數**
- `interesting_only` [bool] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="accessibility-snapshot-option-interesting-only"/><a href="#accessibility-snapshot-option-interesting-only" class="list-anchor">#</a>
  
  從樹中修剪無趣的節點。預設為 `true`。
- `root` [ElementHandle] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="accessibility-snapshot-option-root"/><a href="#accessibility-snapshot-option-root" class="list-anchor">#</a>
  
  快照的根 DOM 元素。預設為整個頁面。

**回傳**
- [NoneType] | [Dict]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="accessibility-snapshot-return"/><a href="#accessibility-snapshot-return" class="list-anchor">#</a>
  - `role` [str]
    
    [角色](https://www.w3.org/TR/wai-aria/#usage)。
  - `name` [str]
    
    節點的人類可讀名稱。
  - `value` [str] | [float]
    
    節點的目前值（如果適用）。
  - `description` [str]
    
    節點的額外人類可讀描述（如果適用）。
  - `keyshortcuts` [str]
    
    與此節點關聯的鍵盤快速鍵（如果適用）。
  - `roledescription` [str]
    
    角色的人類可讀替代方案（如果適用）。
  - `valuetext` [str]
    
    目前值的描述（如果適用）。
  - `disabled` [bool]
    
    節點是否被停用（如果適用）。
  - `expanded` [bool]
    
    節點是否展開或摺疊（如果適用）。
  - `focused` [bool]
    
    節點是否聚焦（如果適用）。
  - `modal` [bool]
    
    節點是否為[強制回應](https://en.wikipedia.org/wiki/Modal_window)（如果適用）。
  - `multiline` [bool]
    
    節點文字輸入是否支援多行（如果適用）。
  - `multiselectable` [bool]
    
    是否可以選擇多個子項（如果適用）。
  - `readonly` [bool]
    
    節點是否為唯讀（如果適用）。
  - `required` [bool]
    
    節點是否為必填（如果適用）。
  - `selected` [bool]
    
    節點是否在其父節點中被選中（如果適用）。
  - `checked` [bool] | "mixed"
    
    核取方塊是否被勾選，或「混合」（如果適用）。
  - `pressed` [bool] | "mixed"
    
    切換按鈕是否被勾選，或「混合」（如果適用）。
  - `level` [int]
    
    標題的層級（如果適用）。
  - `valuemin` [float]
    
    節點中的最小值（如果適用）。
  - `valuemax` [float]
    
    節點中的最大值（如果適用）。
  - `autocomplete` [str]
    
    控制項支援何種自動完成（如果適用）。
  - `haspopup` [str]
    
    目前為節點顯示何種彈出視窗（如果適用）。
  - `invalid` [str]
    
    此節點的值是否以及以何種方式無效（如果適用）。
  - `orientation` [str]
    
    節點是水平方向還是垂直方向（如果適用）。
  - `children` [List]\[[Dict]\]
    
    子節點（如果有且適用）。
