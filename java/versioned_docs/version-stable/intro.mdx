## 簡介

Playwright 專為滿足端對端測試需求而建立。Playwright 支援所有現代渲染引擎，包括 Chromium、WebKit 和 Firefox。在 Windows、Linux 和 macOS 上測試，無論是本機或在 CI 上，支援無頭或有頭模式，並提供原生行動裝置模擬。

Playwright 以一組 [Maven](https://maven.apache.org/what-is-maven.html) 模組分發。使用它最簡單的方法是向專案的 `pom.xml` 加入一個相依項目，如下所述。如果您不熟悉 Maven，請參閱其[文件](https://maven.apache.org/guides/getting-started/maven-in-five-minutes.html)。

## 使用方式

透過安裝 Playwright 並執行範例檔案來開始使用，親自體驗其功能。

```java title="src/main/java/org/example/App.java"
package org.example;

import com.microsoft.playwright.*;

public class App {
    public static void main(String[] args) {
        try (Playwright playwright = Playwright.create()) {
            Browser browser = playwright.chromium().launch();
            Page page = browser.newPage();
            page.navigate("https://playwright.dev");
            System.out.println(page.title());
        }
    }
}
```

使用上面的 Example.java 和 pom.xml，編譯並執行您的新程式：

```bash
mvn compile exec:java -D exec.mainClass="org.example.App"
```

執行此指令會下載 Playwright 套件並安裝 Chromium、Firefox 和 WebKit 的瀏覽器二進制檔案。若要修改此行為，請參閱[安裝參數](./browsers.mdx#install-browsers)。

## 第一個腳本

在我們的第一個腳本中，我們將導航到 `playwright.dev` 並在 WebKit 中截取螢幕截圖。

```java
package org.example;

import com.microsoft.playwright.*;
import java.nio.file.Paths;

public class App {
  public static void main(String[] args) {
    try (Playwright playwright = Playwright.create()) {
      Browser browser = playwright.webkit().launch();
      Page page = browser.newPage();
      page.navigate("https://playwright.dev/");
      page.screenshot(new Page.ScreenshotOptions().setPath(Paths.get("example.png")));
    }
  }
}
```

預設情況下，Playwright 以無頭模式執行瀏覽器。若要查看瀏覽器 UI，請將 [setHeadless](/api/class-browsertype.mdx#browser-type-launch-option-headless) 選項設為 `false`。您也可以使用 [setSlowMo](/api/class-browsertype.mdx#browser-type-launch-option-slow-mo) 來降低執行速度。在偵錯工具[章節](./debug.mdx)中了解更多。

```java
playwright.firefox().launch(new BrowserType.LaunchOptions().setHeadless(false).setSlowMo(50));
```

## 執行範例腳本

```bash
mvn compile exec:java -D exec.mainClass="org.example.App"
```

預設情況下，使用 Playwright 啟動的瀏覽器以無頭模式執行，這意味著執行腳本時不會開啟瀏覽器 UI。若要改變此行為，您可以在啟動瀏覽器時傳遞 `new BrowserType.LaunchOptions().setHeadless(false)`。

## 系統需求
- Java 8 或更高版本。
- Windows 11+、Windows Server 2019+ 或適用於 Linux 的 Windows 子系統（WSL）。
- macOS 14 Ventura 或更新版本。
- Debian 12、Debian 13、Ubuntu 22.04、Ubuntu 24.04，在 x86-64 和 arm64 架構上。
