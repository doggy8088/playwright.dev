## 簡介

Playwright 可以自動產生測試，提供快速開始測試的方式。Codegen 開啟瀏覽器視窗供互動，並開啟 Playwright Inspector 供錄製、複製和管理您產生的測試。

**您將學到**
- [如何錄製測試](/codegen.mdx#recording-a-test)
- [如何產生定位器](/codegen.mdx#generating-locators)

## 執行 Codegen

使用 `codegen` 指令執行測試產生器，接著輸入您想要產生測試的網站 URL。URL 是可選的，如果省略，可以直接在瀏覽器視窗中輸入。

```bash
mvn exec:java -e -D exec.mainClass=com.microsoft.playwright.CLI -D exec.args="codegen demo.playwright.dev/todomvc"
```

### 錄製測試

執行 `codegen` 並在瀏覽器中執行操作。Playwright 會自動為您的互動產生程式碼。Codegen 分析渲染的頁面並推薦最佳定位器，優先使用角色、文字和測試 ID 定位器。當多個元素符合定位器時，產生器會改善它以唯一識別目標元素，減少測試失敗和不穩定性。

使用測試產生器，您可以錄製：
* 動作，例如點擊或填寫，透過與頁面互動
* 斷言，點擊工具列圖示，然後點擊頁面元素來進行斷言。您可以選擇：
  * `'assert visibility'` 斷言元素是可見的
  * `'assert text'` 斷言元素包含特定文字
  * `'assert value'` 斷言元素具有特定值

######

######
完成與頁面互動後，按下 `'record'` 按鈕停止錄製，並使用 `'copy'` 按鈕將產生的程式碼複製到您的編輯器中。

使用 `'clear'` 按鈕清除程式碼並重新開始錄製。完成後，關閉 Playwright Inspector 視窗或停止終端指令。

要了解更多關於產生測試的資訊，請查看我們的詳細指南 [Codegen](./codegen.mdx)。

### 產生定位器

您可以使用測試產生器產生[定位器](/locators.mdx)。
* 按下 `'Record'` 按鈕停止錄製，`'Pick Locator'` 按鈕就會出現
* 點擊 `'Pick Locator'` 按鈕，然後將滑鼠移到瀏覽器視窗中的元素上方，您會看到每個元素下方都高亮顯示了定位器
* 點擊您想要定位的元素，該定位器的程式碼就會出現在 Pick Locator 按鈕旁邊的定位器遊樂場中
* 在定位器遊樂場中編輯定位器來微調它，並在瀏覽器視窗中看到相符的元素高亮顯示
* 使用複製按鈕複製定位器並將其貼上到您的程式碼中

### 模擬

您可以使用模擬為特定檢視區、裝置、顏色方案、地理位置、語言或時區產生測試。測試產生器也可以保持驗證狀態。查看[測試產生器](./codegen.mdx#emulation)指南以了解更多。
