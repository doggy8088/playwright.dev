Playwright 對 Android 自動化有**實驗性**支援。這包括 Chrome for Android 和 Android WebView。

*需求*
* Android 裝置或 AVD 模擬器。
* 執行並與您的裝置已驗證的 [ADB daemon](https://developer.android.com/studio/command-line/adb)。通常執行 `adb devices` 就是您需要做的全部。
* 裝置上已安裝 [`Chrome 87`](https://play.google.com/store/apps/details?id=com.android.chrome) 或更新版本
* 在 `chrome://flags` 中啟用「Enable command line on non-rooted devices」。

*已知限制*
* 尚未支援原始 USB 操作，因此您需要 ADB。
* 裝置需要保持喚醒狀態才能產生螢幕截圖。啟用「保持喚醒」開發者模式會有所幫助。
* 我們沒有針對裝置執行所有測試，因此並非所有功能都能運作。

*如何執行*

Android 自動化腳本的範例如下：

```js
const { _android: android } = require('playwright');

(async () => {
  // Connect to the device.
  const [device] = await android.devices();
  console.log(`Model: ${device.model()}`);
  console.log(`Serial: ${device.serial()}`);
  // Take screenshot of the whole device.
  await device.screenshot({ path: 'device.png' });

  {
    // --------------------- WebView -----------------------

    // Launch an application with WebView.
    await device.shell('am force-stop org.chromium.webview_shell');
    await device.shell('am start org.chromium.webview_shell/.WebViewBrowserActivity');
    // Get the WebView.
    const webview = await device.webView({ pkg: 'org.chromium.webview_shell' });

    // Fill the input box.
    await device.fill({
      res: 'org.chromium.webview_shell:id/url_field',
    }, 'github.com/microsoft/playwright');
    await device.press({
      res: 'org.chromium.webview_shell:id/url_field',
    }, 'Enter');

    // Work with WebView's page as usual.
    const page = await webview.page();
    await page.waitForNavigation({ url: /.*microsoft\/playwright.*/ });
    console.log(await page.title());
  }

  {
    // --------------------- Browser -----------------------

    // Launch Chrome browser.
    await device.shell('am force-stop com.android.chrome');
    const context = await device.launchBrowser();

    // Use BrowserContext as usual.
    const page = await context.newPage();
    await page.goto('https://webkit.org/');
    console.log(await page.evaluate(() => window.location.href));
    await page.screenshot({ path: 'page.png' });

    await context.close();
  }

  // Close the device.
  await device.close();
})();
```

## 方法

### connect {#android-connect}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.28</font><x-search>android.connect</x-search>

此方法將 Playwright 連接到現有的 Android 裝置。使用 [android.launchServer()](/api/class-android.mdx#android-launch-server) 來啟動新的 Android 伺服器實例。

**用法**

```js
await android.connect(wsEndpoint);
await android.connect(wsEndpoint, options);
```

**參數**
- `wsEndpoint` [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-connect-option-ws-endpoint"/><a href="#android-connect-option-ws-endpoint" class="list-anchor">#</a>
  
  要連接的瀏覽器 websocket 端點。
- `options` [Object] *(可選)*
  - `headers` [Object]&lt;[string], [string]&gt; *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-connect-option-headers"/><a href="#android-connect-option-headers" class="list-anchor">#</a>
    
    要與 web socket 連接請求一起發送的額外 HTTP 標頭。可選。
  - `slowMo` [number] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-connect-option-slow-mo"/><a href="#android-connect-option-slow-mo" class="list-anchor">#</a>
    
    將 Playwright 操作減慢指定的毫秒數。這很有用，這樣您就可以看到正在發生的事情。預設為 `0`。
  - `timeout` [number] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-connect-option-timeout"/><a href="#android-connect-option-timeout" class="list-anchor">#</a>
    
    等待建立連接的最長時間（毫秒）。預設為 `30000`（30 秒）。傳遞 `0` 停用逾時。

**回傳**
- [Promise]&lt;[AndroidDevice]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-connect-return"/><a href="#android-connect-return" class="list-anchor">#</a>

### devices {#android-devices}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>android.devices</x-search>

回傳已偵測到的 Android 裝置清單。

**用法**

```js
await android.devices();
await android.devices(options);
```

**參數**
- `options` [Object] *(可選)*
  - `host` [string] *(可選)* <font size="2">Added in: v1.22</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-devices-option-host"/><a href="#android-devices-option-host" class="list-anchor">#</a>
    
    建立 ADB 伺服器連線的可選主機。預設為 `127.0.0.1`。
  - `omitDriverInstall` [boolean] *(可選)* <font size="2">Added in: v1.21</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-devices-option-omit-driver-install"/><a href="#android-devices-option-omit-driver-install" class="list-anchor">#</a>
    
    在附加時阻止自動安裝 playwright 驅動程式。假設驅動程式已經安裝。
  - `port` [number] *(可選)* <font size="2">Added in: v1.20</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-devices-option-port"/><a href="#android-devices-option-port" class="list-anchor">#</a>
    
    建立 ADB 伺服器連線的可選連接埠。預設為 `5037`。

**回傳**
- [Promise]&lt;[Array]&lt;[AndroidDevice]&gt;&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-devices-return"/><a href="#android-devices-return" class="list-anchor">#</a>

### launchServer {#android-launch-server}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.28</font><x-search>android.launchServer</x-search>

啟動客戶端可以連接的 Playwright Android 伺服器。請參閱以下範例：

**用法**

伺服器端：

```js
const { _android } = require('playwright');

(async () => {
  const browserServer = await _android.launchServer({
    // If you have multiple devices connected and want to use a specific one.
    // deviceSerialNumber: '<deviceSerialNumber>',
  });
  const wsEndpoint = browserServer.wsEndpoint();
  console.log(wsEndpoint);
})();
```

客戶端：

```js
const { _android } = require('playwright');

(async () => {
  const device = await _android.connect('<wsEndpoint>');

  console.log(device.model());
  console.log(device.serial());
  await device.shell('am force-stop com.android.chrome');
  const context = await device.launchBrowser();

  const page = await context.newPage();
  await page.goto('https://webkit.org/');
  console.log(await page.evaluate(() => window.location.href));
  await page.screenshot({ path: 'page-chrome-1.png' });

  await context.close();
})();
```

**參數**
- `options` [Object] *(可選)*
  - `adbHost` [string] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-launch-server-option-adb-host"/><a href="#android-launch-server-option-adb-host" class="list-anchor">#</a>
    
    建立 ADB 伺服器連線的可選主機。預設為 `127.0.0.1`。
  - `adbPort` [number] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-launch-server-option-adb-port"/><a href="#android-launch-server-option-adb-port" class="list-anchor">#</a>
    
    建立 ADB 伺服器連線的可選連接埠。預設為 `5037`。
  - `deviceSerialNumber` [string] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-launch-server-option-device-serial-number"/><a href="#android-launch-server-option-device-serial-number" class="list-anchor">#</a>
    
    要啟動瀏覽器的可選裝置序號。如果未指定，當連接多個裝置時會拋出錯誤。
  - `host` [string] *(可選)* <font size="2">Added in: v1.45</font><a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-launch-server-option-host"/><a href="#android-launch-server-option-host" class="list-anchor">#</a>
    
    用於 web socket 的主機。這是可選的，如果省略，當 IPv6 可用時，伺服器將接受未指定的 IPv6 位址 (::) 上的連接，否則接受未指定的 IPv4 位址 (0.0.0.0) 上的連接。建議選擇特定介面來加強安全性。
  - `omitDriverInstall` [boolean] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-launch-server-option-omit-driver-install"/><a href="#android-launch-server-option-omit-driver-install" class="list-anchor">#</a>
    
    在附加時阻止自動安裝 playwright 驅動程式。假設驅動程式已經安裝。
  - `port` [number] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-launch-server-option-port"/><a href="#android-launch-server-option-port" class="list-anchor">#</a>
    
    用於 web socket 的連接埠。預設為 0，會選擇任何可用的連接埠。
  - `wsPath` [string] *(可選)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-launch-server-option-ws-path"/><a href="#android-launch-server-option-ws-path" class="list-anchor">#</a>
    
    服務 Android 伺服器的路徑。為了安全起見，預設為一個無法猜測的字串。
    
    :::warning
    
    任何知道 `wsPath` 的程序或網頁（包括在 Playwright 中執行的那些）都可以取得作業系統使用者的控制權。因此，使用此選項時，您應該使用無法猜測的令牌。
    :::
    
**回傳**
- [Promise]&lt;[BrowserServer]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-launch-server-return"/><a href="#android-launch-server-return" class="list-anchor">#</a>

### setDefaultTimeout {#android-set-default-timeout}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>android.setDefaultTimeout</x-search>

此設定將變更所有接受 [timeout](/api/class-android.mdx#android-set-default-timeout-option-timeout) 選項的方法的預設最長時間。

**用法**

```js
android.setDefaultTimeout(timeout);
```

**參數**
- `timeout` [number]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="android-set-default-timeout-option-timeout"/><a href="#android-set-default-timeout-option-timeout" class="list-anchor">#</a>
  
  最長時間（毫秒）
